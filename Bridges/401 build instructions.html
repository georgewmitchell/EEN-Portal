401 build instructions 

1.Install system fan to chassis with 2 screws 
2.Install power supply (EN-PS-101) with 4 screws
3.Install hard drive 750 GB (EN-DC-100) with 4 screws
4.Install Zotac (EN-BP-009) backplate  with 2 screws
5.Install motherboard Zotac (EN-MB-106) with 2 screws
6.Install Intel processor 1840 (EN-CP-108) onto motherboard line up corner Arrow
7.Install CPU fan (EN-BL-100) snapping 4 pegs and attach cable to motherboard
8.Connect hard drive data cable to motherboard 
9.Connect motherboard power cable to motherboard 
10.Connect hard drive cable to motherboard 
11.Connect system fan to motherboard
12.Connect Black / Yellow ATX power cable to motherboard
13.Use black zip ties to tie off loose cables (follow cabling routing instructions)

Bios setup for 401
Go to Bios features scroll to hard drive bbs priorities
Select Kingston USB drive boot option number one
Press esc
Go to security option choose set up
Go to full screen logo show press space bar to disable
Go to fast boot press space bar to enable
Go to VGA support press and select auto
Go to USB support press full initial
Go to boot mode selection press for legacy
Go to other PCI device ROM priority for legacy
Go to administrator password type in Eagle23soaring
Go to power management
Scroll down to AC back press for memory
Press F10 to save all changes
Program should start building bridge
