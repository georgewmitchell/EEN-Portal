320 build instructions
1.Install power supply (EN-PS-101) with (4) screws into (EN-CH-001)chassis screw down
2.Install system fan screw down front of chassis with (2) screws
3.Install J1900 back plate (EN-BP-008) with (2) screws
4.Install J1900 motherboard (EN-MB-107) onto chassis with (2) screws
5.Install 750 GB hard drive (EN-DC-102) with (4) screws
6.Install 4 GB RAM (EN-ME-105) onto motherboard slot
7.Plug-in power cable (24 pin) to motherboard
8.plug-in sata cable from hard drive to motherboard
9.plug-in system fan onto motherboard
10.plug in sata power cable to hard drive
11.Tie off loose wire with Zip Ties. Make sure follow cabling routing instructions)

Bios:
Go to Advanced setting choose CSM change all settings to "Legacy Only"
Go to Chipset change South Bridge setting change to Restore AC Power Loss button "On"
Go to Security choose administrator password and change to "eagle23soaring"
Go to Boot Screen choose Full Screen Logo Show "Disable"
Go to quick boot and choose "Enable"
Go to Fast Boot choose "Enable"
Go to EFI Driver choose "Auto"
Go to USB Support choose "Full Initial"
Go to boot option choose #1 option
Go to Network Device BBS Priorities choose Verbatim USB driver to install
Hit F4 to save settings
Software installation will start
