303 bridge instructions

1.Install system fan with two screws onto chassis EN-CH-001
2.Install hard drive 750 GB(EN-DC-100)
3.Install J1900 motherboard backplate(EN-BP-008)5
4.Install POE ethernet board backplate(EN-BP-006)
5.Install 9 Port POE switch board(EN-CA-102)
6.Install J1900 motherboard(EN-MB-107) with two screw
7.Install 4 GB RAM(EN-ME-105)onto motherboard
8.Install power supply with black PowerBrick (EN-PS-105) (solder purple/black wire to power supply)
9.Install PCI ethernet blue cable(EN-CA-100) to POE board
10.Install PCI ethernet card to motherboard (glue epoxy)
11.Install 750 GB(EN-DC-100) hard drive data cable to motherboard open data port
12.install system fan to motherboard 
13.install power supply ATX cable to motherboard
14.Epoxy black brick power supply inside of front chassis 
(follow cabling routing instructions)

Bios:
Go to Advanced setting choose CSM change all settings to "Legacy Only"
Go to Chipset change South Bridge setting change to Restore AC Power Loss button "On"
Go to Security choose administrator password and change to "eagle23soaring"
Go to Boot Screen choose Full Screen Logo Show "Disable"
Go to quick boot and choose "Enable"
Go to Fast Boot choose "Enable"
Go to EFI Driver choose "Auto"
Go to USB Support choose "Full Initial"
Go to boot option choose #1 option
Go to Network Device BBS Priorities choose Verbatim USB driver to install
Hit F4 to save settings
Software installation will start
