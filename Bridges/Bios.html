Gigabyte J1900 Motherboard BIOS Settings


Go to Advanced setting choose CSM change all settings to "Legacy Only"
Go to Chipset change South Bridge setting change to Restore AC Power Loss button "On"
Go to Security choose administrator password and change to "eagle23soaring"
Go to Boot Screen choose Full Screen Logo Show "Disable"
Go to quick boot and choose "Enable"
Go to Fast Boot choose "Enable"
Go to EFI Driver choose "Auto"
Go to USB Support choose "Full Initial"
Go to boot option choose #1 option
Go to Network Device BBS Priorities choose Verbatim USB driver to install
Hit F4 to save settings
Software installation will start

!!!UPDATED Jan 2016 J1900 settings!!!

* Set admin bios password to "eagle23soaring"
* Advanced -> CSM -> Network = "Legacy first"
* Advanced -> SIO Misc Functions -> AC Back = "Memory"
* Boot -> Fast Boot = "Enabled"
* Boot -> Fast Boot -> Network Stack driver support = "Enabled"
* Boot Priorities
+ 1. Hard drive (all)
+ 2. USB Flash (any)
+ 3. Realtek PXE NIC #1
+ 4. Realtek PXE NIC #2

Zotac H87 BIOS Settings


Go to Bios features scroll to hard drive bbs priorities
Select Kingston USB drive boot option number one
Press esc
Go to security option choose set up
Go to full screen logo show press space bar to disable
Go to fast boot press space bar to enable
Go to VGA support press and select auto
Go to USB support press full initial
Go to boot mode selection press for legacy
Go to other PCI device ROM priority for legacy
Go to administrator password type in Eagle23soaring
Go to power management
Scroll down to AC back press for memory
Press F10 to save all changes
Program should start building bridge
