302 Build
Remove 302 Kit from cardboard box. 
Store packing materials for later use. 
Remove (4) bottom screws. 
Put screws and tie wraps to the side. 
Take drive plate off (2 screws)
Store power cord for later use. 

Motherboard
Insert 4gb stick EN-ME-105 memory into slot closest to CPU. 
Make sure the tabs click in.
Epoxy battery to motherboard 

Install Hard Drive
Unwrap a 750GB Western Digital HD Attach HD to chassis using 4 screws 
Connect the SATA cable that came with the unit to the drive and motherboard. 
Connect the Hard Drive Power cable to the drive. 
Close up chassis using the 4 screws that were taken out to disassemble the unit

Connect external cables 
Power, Ethernet, monitor, keyboard.
Power On Test - BIOS Settings 
Press power button. 
Make sure LED on. 
Make sure message displayed on monitor 
Press power button to turn off.

Change BIOS: 
Press power button to turn on. 
Press delete during boot to get to BIOS. 
Security>AdminPasswd "eaglemileage" 
Chipset>PCH 10 Configuration>Restore AC Power Loss>ON 
Boot>USB Support>Full Main>Verify 4096MB memory. 
Boot>Boot option #1 >USB Stick

Install Software
Turn unit off. Insert USB Stick. 
Make sure ethernet plugged into WAN
Turn unit on. The software will automatically load onto the hard drive and give serial number. 
Hit Enter, Unit should power off automatically. 
Remove USB key, swap Ethernet to WAN
Power the unit back on.
Start burnin process and let run for 2 days 

Finalize steps
Print an instruction sheet with Attach ID # and place unit on the "Processed" shelf 
Attach Serial # label to the bottom of the unit.
