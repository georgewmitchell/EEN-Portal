302 build instructions
1.Unscrew bottom plate (4) screws
2.Install 750 GB (EN-DC-100) hard drive to bottom plate
3.Install 4 GB RAM (EN-ME-105) onto motherboard
4.Plug in Sata power cable to hard drive
5.Plug-in data cable to hard drive
6.Take out Factory battery and install New battery.
7.Screw back on bottom plate
8.After 302 passes function test Epoxy down battery.
9.Note: Some 302 Chassis's come with a Factory flaw. On left side some will have open port covered with Blue Intel sticker.

Bios
Go to Advanced setting go to ACPI setting choose lock a Legacy "Enable"
Go to Chipset select bios security configuration choose Bios Lock "Enable"
Go to chipset select restore AC power loss choose power on
Go to boot bios choose quiet boot enable
Go to fast boot choose enable
Go to VGA support select auto
Go to USB support select full initial
Go to CSM parameters choose boot option filter legacy only
Go to launch storage up ROM policy choose legacy only
Go to launch video up ROM policy select legacy only
Go to other PCI device ROM priority select legacy oprom 
Go to hard drive BBS priorities select verbatim storage and go boot option #1
F10 save and exit
